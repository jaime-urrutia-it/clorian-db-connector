# Clorian DB Connector

## Descripción
Conector Java desarrollado para la integración y gestión de conexiones con bases de datos MySQL. Este módulo forma parte de un sistema más amplio de automatización de procesos y gestión de incidentes.

## Tecnologías
- **Lenguaje:** Java
- **Base de Datos:** MySQL
- **Librerías:** JSON (org.json)
- **Entorno:** Eclipse IDE

## Estructura del Proyecto
- `src/`: Código fuente principal.
- `lib/`: Dependencias externas (JARs).
- `sql-scripts/`: Scripts SQL para inicialización y consultas.
